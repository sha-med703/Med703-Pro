# 🎓 Med703 Pro

> AI 驱动的医学考研智能学习助手

---

## 📖 项目简介

Med703 Pro 是一款专为医学考研打造的智能学习管理系统。

项目主要用于帮助考研学生进行：

- 📚 学习计时
- 🍅 番茄钟
- 📈 学习统计
- 🔥 GitHub 风格学习热力图
- 📅 艾宾浩斯复习
- 🎯 每日学习目标
- ⚙️ 个性化设置
- 🤖 AI 学习教练（开发中）

---

# 🚀 快速开始

安装依赖：

```bash

```

启动项目：

```bash
npm run dev
```

打包：

```bash
npm run build
```
